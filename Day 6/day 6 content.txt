Step to be followed when a new windows machine is created:
----------------------------------------------------------
- install the windows OS (server/client)
- give the password for the administrator (default 1st user in windows)
- change
	- computer name (do not restart)
	- IP address
	- time zone
- install VMWare tools
- after tools installation, restart the virtual machine
- login to the VM and take the snapshot of the VM.

to install data deduplication
open powershell
	> Install-WindowsFeature -Name FS-Data-Deduplication
	
NFS sharing
	- Network File Sharing
	- it is used for sharing the data between similar OSs
	- like:
		- windows to windows
		- linux to linux

Samba server
	- it is used for sharing the data on cross-platforms
	- like:
		- linux to windows

Windows Server Backup (WSB)
------------------------------
- WSB is a built-in tool for taking backups and restoring them if needed.
- backup can be taken as:
	1. full server backup
	2. custom backup
